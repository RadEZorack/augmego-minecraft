# avatar-core

Tiny shared Java module for loading a static GLB into a neutral avatar model.

## Scope

- GLB parsing only
- Neutral mesh data only
- No Minecraft, Fabric, or renderer-specific code
- No animation, skeletons, morphs, uploads, or networking

## Public API

- `AvatarModel`
- `AvatarMesh`
- `AvatarLoader`
- `AvatarLoadException`
